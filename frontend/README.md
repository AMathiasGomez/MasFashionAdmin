# Frontend

Aplicacion Angular para el sistema administrativo del negocio de ropa femenina.

## Stack

- Angular
- Bootstrap
- Bootstrap Icons
- Chart.js

## Configuracion

La API esperada por defecto es:

```txt
http://localhost:3000/api
```

El valor esta en:

```txt
src/app/core/config/api.config.ts
```

## Comandos

```bash
npm install
npm start
npm run build
```

La aplicacion queda disponible en:

```txt
http://localhost:4200
```

## Modulos iniciales

- Login con JWT
- Layout administrativo con sidebar y navbar
- Dashboard con Chart.js
- Productos
- Categorias
- Proveedores
- Inventario
- Pedidos
- Clientes
- Insumos
- Finanzas
- Reportes
- Descarga de reportes PDF y Excel
